# Dhurandham Fighters – VR Fire Evacuation Training

## Project Overview

**Dhurandham Fighters** is a Virtual Reality (VR) training project that helps people learn how to escape safely during a fire emergency.

In real life, when a fire happens in buildings like malls, offices, or hostels, people often panic and do not know what to do. This project creates a **virtual environment of a building on fire**, where users can practice the correct way to evacuate.

The goal is to teach basic safety actions such as staying low in smoke, avoiding elevators, and following exit directions.

## Why This Project

Many accidents happen because people are not trained for emergencies. VR technology allows people to experience a dangerous situation in a **safe and controlled environment**.

Using this simulation, users can:

* Understand how fires spread
* Learn how smoke affects visibility
* Practice finding the nearest exit
* Learn why elevators should not be used during fire emergencies

## Technologies Used

This project is built using simple web technologies:

* HTML
* CSS
* JavaScript
* A-Frame for creating the 3D VR environment

The advantage of using A-Frame is that the simulation can run **directly in a web browser** without installing any software.

## Features

* 3D building environment
* Fire and smoke simulation
* Direction arrows showing evacuation route
* Warning messages (for example: do not use elevator)
* Staircase exit system
* Safety instructions during evacuation

## How to Run the Project

1. Download or clone this repository.

2. Open the project folder.

3. Run the main file in your browser:

fire-evacuation-vr.html

4. Click inside the screen to start the simulation.

## Controls

* Mouse → Look around
* W A S D → Move inside the building
* Follow arrows to find the exit
